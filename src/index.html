configuration :
   1) const video = new VideoTile('.videocon') ---videocon is the selectors of video.
   2) data-title, data-url, data-caption must be supplied for every tile. otherwise it wont work.
   3) does not support built in lazy loading. but you can refer video-config.ts to see how to achieve that.
   4) after initialization you need to use video.render() to open modal ---refer video-config.ts
   5) supports both dynamic and static tiles.
